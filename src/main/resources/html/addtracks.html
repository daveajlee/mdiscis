Add Track(s)

To add new tracks to a disc, click on "Add Track(s)" at the bottom of the screen. The dialog shown above will be appear on screen. You should enter the following information:
Disc: Choose from the drop-down list which disc you would like to add the tracks to.
Start Track: Choose the track number to start from. This can be any number between 1 and 50.
End Track: Choose the end track number. This can be any number equal to or greater than the start track number and less than or equal to 50. If it equals the start track, only one track will be added. If it is less than the start track, an error message will be shown.
Subject: Enter the subject.
Speaker: Enter the speaker.
Talk Title Enter the talk title.
Date: Choose the date of the track(s) using the arrows. Today's date is shown by default.
Recorded? Click on the checkbox if the track(s) have been transferred to PC.

Finally, click on the "OK" button to add the track(s). A confirmation message will appear in the status bar at the bottom of the main screen.
