Load Disc Store

To load a disc store, click on the "File" menu in the menu bar and click "Load". The dialog box shown in the picture above will be displayed on screen. All MDISCIS Index Files are shown in the selected file. Click on the one you want to open and click "Open". The contents of the first disc will then be displayed in the main window. If you attempt to open a file which is not a valid MDISCIS Index File, you will receive an error message.
